# ArrListLab

### Реализация:
- Реализован шаблонный класс `TArrList<T>`
- Методы имеют асимптотику:
  - `PushStart`, `PushEnd`, `PopStart`, `PopEnd`, `IsEmpty`, `IsFull` - O(1)
  - Остальные методы O(n)
- Реализованы конструкторы
- Реализован деструктор
- Перегружены операторы: `+`, `+=`, `==`, `!=`, `[]`, `<<`, `>>`
- Реализован итератор
- Реализованы методы из шаблонного класса:
  - `Print()`
  - `Resize()`
  - `GetFreeNodeIndex()` — взять свободный индекс из пула
  - `freeIndex(idx)` — вернуть индекс в пул свободных
- методы для доп задания:
  - `InsertAfter(idx, value)`
  - `Remove(idx)`
  - `Count(value)` — количество вхождений
  - `FindAll(value, result)` — все физические индексы с заданным значением
